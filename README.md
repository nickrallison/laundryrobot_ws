# laundryrobot_ws
## To initialize the driver 
Copy the contents of this folder to a sub directory called src

```
laundryrobot_ws
│
└───folder1
|   │   files...
│   
└───folder2
    │   files...
```

Change top to bottom

```
laundryrobot_ws
│
└───src
    │
    └───folder1
    |   │   files...
    | 
    └───folder2
        |   files...

```
